───────────────────────────────────────────────────────────
SCHRITT 1: Git-Tool öffnen
───────────────────────────────────────────────────────────

1. Klicke LINKS in Replit auf das Symbol: 🔧 (Tools)
2. Falls Git noch nicht da ist:
   → Klicke auf das "+" Symbol
   → Wähle "Git" aus der Liste
3. Das Git-Panel öffnet sich

───────────────────────────────────────────────────────────
SCHRITT 2: Mit GitHub verbinden (falls noch nicht verbunden)
───────────────────────────────────────────────────────────

1. Im Git-Panel: Klicke "Connect to GitHub"
2. Logge dich bei GitHub ein
3. Erlaube Replit den Zugriff

───────────────────────────────────────────────────────────
SCHRITT 3: Alle Dateien hochladen
───────────────────────────────────────────────────────────

1. Im Git-Panel siehst du eine Liste mit geänderten Dateien
2. Klicke "Stage All" (WICHTIG: Alle Dateien müssen dabei sein!)
3. Schreibe in das Textfeld:
   "Add all deployment files for Render"
4. Klicke "Commit"
5. Klicke "Push"

✅ FERTIG! Code ist auf GitHub!
